About the project


To provide a real-world Task Manager Application that users can use to manage various tasks such as create, edit, delete, and complete tasks. Existing users can login to their accounts while new users can create new accounts. This web application is responsible for day-to-day planning, schedules, and resources.


With a user-friendly interface and features such as task list creation, editing and updating lists and so on, our web app streamlines the task management process and increases productivity. Whether you're working on personal projects or collaborating with a team, our web app provides a centralized platform to keep on track and focus on the tasks to be completed.
